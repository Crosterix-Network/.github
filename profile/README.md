## 🧭 Overview
Crosterix builds **privacy utilities** for the Ethereum ecosystem.  
We focus on user-level privacy, stealth transfers, and zero-knowledge access for Web3 applications.

### 🪩 Core Products
| Module | Description |
|---------|--------------|
| 🕶️ **PrivatePay** | Stealth transactions between EVM wallets using ephemeral keys and encrypted memo fields. |
| 🔄 **PrivateSwap** | Privacy-preserving DEX aggregator for ERC-20 assets. |
| 🧠 **ZK Access Layer** | ZK-gated authentication for dApps, ensuring user anonymity. |
| 📂 **Encrypted Links** | On-chain encrypted file sharing powered by Lit Protocol & IPFS. |
