## What is Crosterix?
Crosterix builds **privacy utilities** for Solana & EVM:
- **PrivatePay / PrivateSwap** – anonymized transfers & DEX routing with privacy guards  
- **Encrypted File Links** – shareable links secured with on-chain gating  
- **Privacy Relayer & ZK Access Layer** – hide IP/wallet identity and gate features using proofs
